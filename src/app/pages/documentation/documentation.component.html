 vpn_key API Keys Setup How to obtain and configure your AI API keys

Google Gemini

Get your free API key from Google AI Studio:

  1. Visit Google AI Studio
  2. Sign in with your Google account
  3. Click "Get API Key" and create a new key
  4. Copy the key and add it to your profile

OpenAI

Get your API key from OpenAI platform:

  1. Visit OpenAI Platform
  2. Create an account and add billing information
  3. Go to API Keys section
  4. Create a new secret key and copy it

Mistral AI

Get your API key from Mistral platform:

  1. Visit Mistral Console
  2. Create an account and verify email
  3. Navigate to API Keys
  4. Generate a new key and copy it

OpenRouter

Access multiple models through OpenRouter:

  1. Visit OpenRouter
  2. Sign up and add credits
  3. Go to Keys section
  4. Create a new API key
 help Troubleshooting Common issues and solutions Application Won't Start

Possible causes and solutions:

  • Check if you're logged in with valid credentials
  • Ensure your internet connection is stable
  • Verify that your account has remaining time/credits
  • Try restarting the application
  • Check Windows Defender or antivirus settings
 AI Not Responding

Check the following:

  • Verify your API keys are correctly configured
  • Check if you have sufficient API credits
  • Ensure your internet connection is working
  • Try a different AI model from the settings
  • Check for any error messages in the application
 Hotkeys Not Working

Try these solutions:

  • Run the application as administrator
  • Check if other applications are blocking hotkeys
  • Restart the application
  • Verify the application has focus
  • Check Windows accessibility settings
